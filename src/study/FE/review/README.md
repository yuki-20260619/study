# 要復習ノート

基本情報技術者試験で理解が曖昧だった内容や、繰り返し間違えた内容をまとめています。

## 目的

- 苦手分野の克服
- 試験直前の復習
- 間違えやすいポイントの整理

---

## 記録する内容

- 間違えた問題
- 覚えにくい用語
- 計算問題
- 混同しやすい知識
- 解き方のポイント

---

## 作成ルール

テーマごとに1ファイルで管理します。

例

```text
review/
├── project-management.md
├── database.md
├── network.md
├── security.md
└── algorithm.md
```

必要に応じて内容を追記・更新し、知識を蓄積していきます。
