# 基本情報技術者試験

基本情報技術者試験の学習記録です。

## 学習目的

- 基本情報技術者試験合格
- コンピュータサイエンスの基礎理解
- 開発・インフラ両方の知識習得
- 実務で活かせるIT知識の定着

---

## ディレクトリ構成

### notes

教材を学習しながら整理した学習ノートです。

内容

- 学習内容のまとめ
- 図や表による整理
- 練習問題
- 解説
- 振り返り

```text
notes/
```

---

### practice

過去問演習の記録です。

内容

- 過去問演習
- 正答率
- 学習時間
- 学習範囲

```text
practice/
```

---

### review

復習が必要な内容をテーマごとに整理します。

内容

- 苦手分野
- 間違えやすいポイント
- 覚え方
- 試験直前の復習用まとめ

```text
review/
```

---

## 学習方針

1. notesで知識を整理する
2. practiceで理解度を確認する
3. reviewで苦手分野を復習する
